# 📖 고구마 7남매 시리즈 웹 앱 가이드

이 문서는 현재 구축된 '고구마 7남매' 웹 동화책 애플리케이션의 구조와 관리 방법을 설명합니다.

---

## 1. 🌐 웹사이트 구성 및 흐름 (Navigation Flow)

사용자가 웹사이트에 접속했을 때의 경험은 **실제 책을 펼치는 과정**과 동일하게 설계되었습니다.

| 순서 | 페이지 이름 | URL | 설명 |
| :--- | :--- | :--- | :--- |
| **1** | **메인 홈 (시리즈 표지)** | `http://localhost:3000/` | 책의 겉표지입니다. 시리즈 전체의 **대표 이미지**와 **작가의 말**이 보입니다. '목차 보기' 버튼을 눌러 들어갑니다. |
| **2** | **시리즈 목차 (로드맵)** | `http://localhost:3000/series` | 책의 앞부분에 있는 목차 페이지입니다. 1권부터 5권까지의 **전체 로드맵**을 보여줍니다. |
| **3** | **동화 읽기 (뷰어)** | `http://localhost:3000/vol/1` | 실제 동화책 본문입니다. 1권(엄마의 선물)의 내용이 표지부터 뒷표지까지 펼쳐집니다. |

---

## 2. 📂 주요 파일 및 역할 (File Structure)

이 프로젝트는 **Next.js**로 만들어졌으며, **Markdown(.md)** 파일을 수정해서 내용을 바꿀 수 있는 구조입니다.

### 📝 콘텐츠 파일 (Content Data)
가장 자주 수정하게 될 파일들입니다. `content` 폴더 안에 있습니다.

- **`content/story.md` (시리즈 본부)**
  - **역할**: 시리즈 전체의 정보를 담고 있습니다.
  - **포함 내용**:
    - `## 시리즈 표지`: 메인 홈에 나올 이미지와 텍스트(작가의 말).
    - `## 📊 시리즈 로드맵`: 목차 페이지에 나올 권별 정보.
    - `캐릭터 소개`: (참고용 데이터)
  - **수정 방법**: 작가의 말을 바꾸거나, 2권/3권 제목을 바꿀 때 이 파일을 엽니다.

- **`content/vol1.md` (1권 본문)**
  - **역할**: 1권 '엄마의 선물'의 실제 이야기 내용입니다.
  - **포함 내용**: 각 장면별 제목, 이미지 경로, 텍스트.
  - **수정 방법**: 1권의 대사를 고치거나 오타를 수정할 때 이 파일을 엽니다.

### 💻 시스템 파일 (System Code)
개발적인 로직이 들어있는 파일들입니다.

- **`lib/story-loader.ts` (데이터 로더)**
  - **역할**: 마크다운 파일(`story.md`, `vol1.md`)을 읽어서 웹사이트가 이해할 수 있는 데이터로 바꿔주는 **번역기** 역할을 합니다.
  - **핵심 기능**:
    - 파일 내용을 읽어 '장면(Scene)' 단위로 쪼갭니다.
    - `(21페이지)` 같은 제목을 보고 자동으로 이미지 파일명(`020.png`)을 찾아줍니다.
    - `story.md`에서 '시리즈 표지' 정보를 쏙 뽑아내 메인 페이지에 전달합니다.

- **`app/page.tsx` (메인 홈 화면)**
  - **역할**: `http://localhost:3000/` 접속 시 보이는 첫 화면을 그립니다.
  - `story.md`의 '시리즈 표지' 데이터를 가져와 예쁘게 보여줍니다.

- **`app/series/page.tsx` (목차 화면)**
  - **역할**: `http://localhost:3000/series` 접속 시 보이는 목록 화면을 그립니다.
  - `story.md`의 '로드맵' 데이터를 가져와 리스트로 보여줍니다.

- **`app/vol/[id]/page.tsx` (읽기 화면)**
  - **역할**: `http://localhost:3000/vol/1` 접속 시 보이는 동화책 뷰어입니다.
  - `vol1.md` 파일을 읽어와서 책처럼(좌측 이미지, 우측 텍스트) 보여줍니다.

---

## 3. 🎨 이미지 관리 (Images)

이미지는 `public/images` 폴더에 분류되어 있습니다.

- **`public/images/series_cover.png`**: 메인 홈에 나오는 **시리즈 대표 표지**입니다.
- **`public/images/vol1/`**: 1권에 들어가는 모든 이미지가 여기 있습니다.
  - `cover_front.png`: 1권 앞표지
  - `cover_back.png`: 1권 뒷표지
  - `002.png`, `004.png`... : 각 펼침면(Spread)의 이미지들입니다. (짝수 번호 사용)

---

## 4. ✨ 기능 요약 (Features)

1. **자동 이미지 매칭**:
   - `장면 1 (2-3페이지)`라고 쓰면 자동으로 `002.png`를 찾아 보여줍니다.
   - 홀수 페이지(21p)로 시작하는 장면도 자동으로 짝수 번호(20p, `020.png`) 이미지를 찾도록 똑똑하게 만들어졌습니다.

2. **전체 읽어주기 (TTS)**:
   - 1권 표지 페이지 우측 상단에 **[전체 읽어주기]** 버튼이 있습니다.
   - 누르면 처음부터 끝까지 책을 자동으로 넘기며 읽어줍니다.
   - 각 페이지마다 따로 읽고 싶으면 텍스트 옆의 스피커 아이콘 🔊을 누르면 됩니다.

3. **반응형 디자인**:
   - PC에서는 책을 펼친 모양으로, 모바일에서는 위아래 스크롤 형태로 보기 편하게 바뀝니다.
